# machinic-wip

_work-in-progress visualization functions_

## Installation

Pip:

```
pip3 install git+https://github.com/galencm/machinic-wip --user
```

Develop while using pip:

```
git clone https://github.com/galencm/machinic-wip
cd machinic-wip/
pip3 install --editable ./ --user
```
